# ❤️ Heart Disease Predictor

A Machine Learning web application that predicts the likelihood of heart disease based on a patient's medical parameters. The project uses a trained K-Nearest Neighbors (KNN) model and provides predictions through a simple and interactive web interface.

---

## 🚀 Features

- Predicts the likelihood of heart disease.
- User-friendly web interface.
- Real-time predictions.
- Data preprocessing using feature scaling.
- Trained KNN Machine Learning model.
- Input validation for medical parameters.
